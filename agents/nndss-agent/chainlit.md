# Australian Disease Surveillance Agent

Ask questions about notifiable disease data from the **National Notifiable Diseases Surveillance System (NNDSS)**.

### What I can do
- Query notification counts by disease, state, and year
- Calculate per-capita rates using ABS population data
- Compare trends across diseases and states
- Explain surveillance methodology and data limitations

### Data available
- **4 diseases** in detail: Influenza, Meningococcal, Pneumococcal, Salmonellosis (2008-2025)
- **73 diseases** via fortnightly reports (2024-2026)
- **Population data** for all states/territories (ABS ERP 2008-2025)

> Powered by Trino SQL + LangGraph + MLflow tracing
